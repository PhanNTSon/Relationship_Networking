# Relationship Networking Project

## Introduction
This is a self-project which I choose just for fun.
The main goal here is to improve my skill in CI/CD, IaC with Terraform and system design.
The Topic of this project is: **a network showing the relationship between person to other person**.

## Tech stack
**Frontend**: React + Vite
**Backend**: Javascript (use in *Lambda*)
**Infa**: AWS Cognito, API Gateway, Lambda, VPC, RDS, S3
